# NyanTimer
Stackmat Timer for cubing



## 概要

自作スタックタイマー

### 機能

標準のスタックタイマーの機能(信号出力を含む)

インスペクションタイム計測(秒数を知らせる機能を含む)

ラップタイム計測(99個まで計測可能)

## 開発環境

Visual Studio Code

Arduino IDE

Autodesk Eagle

Autodesk Fusion 360
